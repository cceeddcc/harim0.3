상세 설명

CSS는 html 문서를 좀더 깔끔하고 효율적으로 스타일링하는데 사용됩니다. font size,color,align 등으로 폰트의 크기와 색깔, 위치 등을 지정할 수 도 있고 font-decoration을 통해 밑줄을 그을 수도 데코를 없앨 수도 있습니다. border 속성을 통해 경계선을 그을 수 있고 margin 과 padding 등으로 크기를 미세하게 조절할 수 있습니다. 오른쪽 클릭후 검사를 들어가면 좀더 빠르고 시각적으로 활용하는데 도움이 됩니다. grid 속성을 통해 화면 레이아웃을 구성할 수 있습니다. 그런데 style태그 가 굉장히 길어지는데 html문서 하나하나를 조금씩 바꾸기엔 너무 힘들어진다. 이러한 중복을 제거하기 위해 style 태그안에 있는 코드를 따로 .css 파일로 만든 뒤 style태그가 있던 자리에 link 태그를 사용해서 href 로 .css파일을 가져오는 것으로 하면 훨씬 효율적으로 사용할 수 있다.
